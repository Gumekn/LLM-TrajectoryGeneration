### 数据流转设计

```
Waymo原始pkl文件
      ↓
[Step 1] 数据加载模块 → 输出: ScenarioData对象（字典格式）
      ↓
[Step 2] 风险计算模块 → 输出: 带RiskScore的ScenarioData
      ↓
[Step 2] 片段截取模块 → 输出: 危险片段数据（KeyVehicleTrajectory_Risk）
      ↓
[Step 3] 特征提取模块 → 输出: 12维特征向量 + 片段数据
      ↓
[Step 4] 穷举生成模块 → 输出: 候选轨迹列表（1000条，含生成参数）
      ↓
[Step 5] 物理验证模块 → 输出: (通过列表, 剔除列表) + 验证报告
      ↓
[Step 3] 特征再提取 → 输出: 通过验证的候选轨迹特征
      ↓
[Step 6] RAG检索模块 → 输出: 每条候选的相似案例列表
      ↓
[Step 7] LLM评估模块 → 输出: 评估结果（合理/不合理 + 置信度）
      ↓
[Step 8] 人工审核模块 → 输出: 审核记录 + 知识库更新
      ↓
[Step 9] CARLA适配模块 → 输出: JSON/XOSC格式场景文件
```
